# Reservation-Cancellation-Prediction

## Project Overview
This project predicts whether a hotel reservation will be canceled using machine learning. The model helps hotels optimize resource allocation and revenue management by accurately forecasting cancellations.

## Problem Statement
Online hotel reservation platforms have led to increased cancellations and no-shows, creating operational challenges for hotels. This service predicts cancellation probability using historical booking data.

## Features
- Data preprocessing and feature engineering
- Logistic Regression model with feature selection
- REST API for predictions
- Docker containerization
- Model persistence and versioning
